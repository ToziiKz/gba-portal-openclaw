# GBA Portal Workspace

Ce repository contient le code source du portail web du Groupement Bruche Ackerland (GBA).

## Structure du Projet

Le projet principal est situé dans le dossier `gba-portal/`.
Il s'agit d'une application Next.js 15 (App Router) utilisant Tailwind CSS v4.

### Commandes Rapides

- **Démarrer le serveur de développement :**
  ```bash
  cd gba-portal
  npm run dev
  ```
  L'application sera accessible sur [http://localhost:3000](http://localhost:3000).

- **Linter le code :**
  ```bash
  cd gba-portal
  npm run lint
  ```

- **Construire pour la production :**
  ```bash
  cd gba-portal
  npm run build
  ```
